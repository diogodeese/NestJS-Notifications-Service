## Description

Notification microservice example built with [Nest](https://github.com/nestjs/nest).

### Features

* Send Notification
* Read Notification
* Unread Notification
* Cancel Notification
* Count Recipient Notifications
* Get Recipient Notifications

## Getting Started

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Envoirement Variables

```shell
DATABASE_URL="file:./dev.db"
```
